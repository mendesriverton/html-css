Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag B (não semântica).

Nesta frase, temos um termo em destaque utilizando a tag STRONG (semântica).

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag I (não semântica).

Nesta frase, temos um termo em destaque usando a tag EM (semântica).

Texto marcado

Nesta frase, temos um termo marcado usando a tag MARK.

Neste parágrafo, temos outro texto marcadono final utilizando a tag MARK.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo com as tag's BIG (obsoleto) e SMALL (não obsoleto).

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, com a tag DEL.

Texto inserido

Podemos marcar um texto como inserido para indicar que ele foi adicionado depois com a tag INS.

Existe também o texto sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x10+3 utilizamos a tag SUP.

Texto subescrito

Para inserir coisas do tipo H20 utilizamos a tag SUB.
